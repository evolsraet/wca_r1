# 개발환경

- 백엔드프레임웍 : laravel 10.
- 프론트기술스택 : Blade, Alpine Js, Bootstrap 5.3~
- 기존 백엔드 API 그대로 사용
- sanctum 사용

# 개요

- 기존 프론트엔드
    - vuejs 를 사용해 개발
    - /v1 prefix 로 교청
- 새 프론트엔드
    - /v2 prefix
    - '프론트기술스택'을 활용하고 '기존 프론트엔드' 를 참고해 빠르게 개발

# 구조
    - 폴더
        - assets : /resources/v2/ js,img,sass 폴더 활용
        - views : /resources/views/v2 사용
